void op_addcmul_cpu ( CTHOperator *  op)

out = input + multiplier * tensor_1 * tensor_2 ref: https://pytorch.org/docs/stable/torch.html#torch.addcmul

Op requirement:

  • # of input tensors: 3
    • input: the input always at index 0
    • tensor_1: tensor with name tensor_1
    • tensor_2: tensor with name tensor_2
  • # of arguments: 1
    • CTH_PARAM_TYPE_MULTIPLIER
  • # of output tensors: 1
    • output: the input always at index 0

Note: does not support boradcast
