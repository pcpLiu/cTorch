void op_argmin_cpu ( CTHOperator *  op)

Returns the indices of the minimum values of a tensor across a dimension.

When there's multiple min values in a dim, it returns the last position
it meets.
Note
In this implementation, keepdim is always false.
Parameters
opInputs & Outputs & Params:
  • # of inputs: 1
  • # of outputs: 1
    • Output tensor type should be CTH_TENSOR_DATA_TYPE_INT_64
  • Argument:
    • dim (int): the dimension to reduce. If -1, the argmin of the flattened input is returned.
