CTHScheduler* cth_new_scheduler ( CTHConfig *  config,
CTHGraph *  graph 
)

Create a new scheduler based on config and graph. Function will parse graph's nodes and fill into job_list.

Arguments

  • config: execution config
  • graph: a computation graph

Fill job_list.

Note: for each node's node_id in this graph, scheduler will reassign them from 0 - N. The queue_status, done_status & ready_status have a assumption that all nodes' node_id are natural numbers starting from 0.
