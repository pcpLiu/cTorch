void op_logsumexp_cpu ( CTHOperator *  op)

Returns the log of summed exponentials of each row of the input tensor in the given dimension dim. The computation is numerically stabilized.

Op requirement:
  • # of input tensors: 1
  • # of arguments: 1
    • CTH_PARAM_TYPE_DIM
  • # of output tensors: 1
    • The output tensor data type should be floating
Note
keepdim is always false as it setup in PyTorch
