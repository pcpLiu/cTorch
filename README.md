# Introduction

cTorch is an inference engine for PyTorch models implemented with C.
It was developed for project [cTorch.js]().

# Depedencies

cTorch could use one of the following backends:

- OpenBLAS (default, coming with the lib)
- Intel MKL
- Accelerate (Mac OS)
- CUDA
